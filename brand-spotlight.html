Daydream Brand Email Builder

Fill in the campaign details, create a new brand folder inside Customer.io Assets folder 741, upload assets using the recommended nomenclature, paste the asset URLs, then copy the generated code into the Brand Template in Customer.io. Eyebrow, footer, social links, spacing, and structure stay fixed.

Email inputs

Workflow
1. Create a new brand folder in Customer.io Assets folder 741.
2. Upload the hero and PDP images using the generated filenames below.
3. Paste the Customer.io asset URLs into this builder.
4. Copy the generated code and paste it into the Customer.io Brand Template code editor.

Used for both black CTAs: Shop Brand Name

Generated Asset Convention
Customer.io folder 741 → brand-name
Hero image → brand-name-hero.jpg
PDP images → brand-name-01.jpg, brand-name-02.jpg, brand-name-03.jpg

Recommended filename: brand-name-hero.jpg

Asset naming standard
• Lowercase only
• Convert spaces to hyphens
• Convert "&" to "and"
• Keep numbers: Brand Name → brand-name
• Avoid item names in filenames; use the PDP name field for alt text

PDPs

Each PDP needs an image asset URL, PDP link, and alt text/item name. Output automatically renders 2 PDPs per row.
