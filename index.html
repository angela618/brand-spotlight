Daydream Brand Spotlight Builder

Build a Brand Spotlight email without editing the template manually. Update imagery in Figma, upload assets to Customer.io, paste asset URLs and PDP details here, then copy the generated code into the duplicated Customer.io campaign.

Email inputs

Workflow
  1. Update the hero image and PDP images in the Brand Spotlight Figma.
  2. Create a new brand folder in Customer.io Assets folder 741.
  3. Upload the hero and PDP images using the generated filenames below.
  4. Paste the Customer.io asset URLs and PDP details — link + alt text — into this builder.
  5. Make a copy of the audience segment.
  6. Make a copy of the Brand Spotlight campaign, then replace the audience segment and paste in the generated email code.

Used for both black CTAs: Shop Brand

Generated Asset Convention
Customer.io folder 741 → brand-name
Hero image → brand-name-hero.jpg
PDP images → brand-name-01.jpg, brand-name-02.jpg, brand-name-03.jpg

Recommended filename: brand-name-hero.jpg

Asset naming standard
• Lowercase only
• Convert spaces to hyphens
• Convert "&" to "and"
• Keep numbers: 16Arlington → 16arlington
• Avoid item names in filenames; use the PDP name field for alt text
• PDP images render at width="100%" automatically

PDPs

Recommended: 6–8 PDPs for best visual balance. The builder automatically renders 2 PDPs per row.
